# TraceSearchEngine

A high-performance, AI-powered developer search engine built with Next.js, Express, FlexSearch, Puppeteer, and NVIDIA NIM.

## Features
- Search-as-you-type with instant results
- AI-powered answers via NVIDIA NIM (LLaMA 3.1)
- Full-text search with multi-signal relevance ranking
- Web crawler with per-domain deduplication
- Premium dark-mode UI with Framer Motion animations

## Stack
- **Frontend**: Next.js 15, TypeScript, Framer Motion
- **Backend**: Express, FlexSearch, Puppeteer, Cheerio
- **Database**: Neon Postgres (Serverless)
- **AI**: NVIDIA NIM API (LLaMA 3.1 70B)
